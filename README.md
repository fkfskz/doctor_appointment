# Сервис записи на прием к врачу.

## Описание
Веб-сервис для самостоятельной записи пациентов на прием к врачам городской поликлиники №1.

## Функции
- Просмотр списка врачей
- Выбор свободного времени
- Запись на прием
- Проверка записей по телефону
- Отмена записи

## Технологии
- Python 3.8+
- Flask
- JSON (хранение данных)

## Установка и запуск
```bash
pip install flask
python app.py
